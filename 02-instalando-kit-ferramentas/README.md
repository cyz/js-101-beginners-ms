# Instalando tu kit de herramientas

Para empezar a desarrollar código en JavaScript en [Node.js](https://nodejs.org/), necesitarás algunas herramientas. En particular, necesitarás instalar Node.js y un editor de código.

- [Descarga de Visual Studio Code](https://code.visualstudio.com?WT.mc_id=javascript-34431-gllemos)
- [Tutorial: Configurando tu entorno de desarrollo Node.js directamente en Windows](https://docs.microsoft.com/windows/nodejs/setup-on-windows?WT.mc_id=javascript-34431-gllemos)
- [Tutorial: Configura tu entorno de desarrollo Node.js con WSL 2](https://docs.microsoft.com/windows/nodejs/setup-on-wsl2?WT.mc_id=javascript-34431-gllemos)
- [Versión de NVM para Linux, macOS o WSL](https://github.com/nvm-sh/nvm/blob/master/README.md)
- [Versión de NVM para Windows](https://github.com/coreybutler/nvm-windows)
